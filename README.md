## ▶️ How to Run
1. Install Java (JDK 8 or above)
2. Clone the repository:
   git clone <your-repo-link>

3. Navigate to the project folder:
   cd project-folder

4. Compile the program:
   javac Main.java

5. Run the program:
   java Main
